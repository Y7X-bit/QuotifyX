# 💬 QuotifyX

An AMOLED-themed quote app with glowing red buttons, built in Python using `customtkinter`.

---

### ⚡ Features
- 🔥 Red-glow aesthetic, AMOLED UI
- 🎯 Get quotes (API + local)
- 💾 Save favorites
- 🎤 Text-to-speech
- 📋 Copy to clipboard
- 🖼️ Export quote as image
- ➕ Add your own quotes

---

### 🚀 Run It

```bash
pip install -r requirements.txt
python quotifyx.py
