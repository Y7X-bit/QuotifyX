## 🌟 Features at a Glance

🎯 **Get Quotes** from multiple categories (Motivation, Funny, Productivity, API)  
🔊 **Text-to-Speech** with `pyttsx3`  
📋 **Copy to Clipboard**  
💾 **Save Favorites** to memory  
🖼️ **Export as Image**  
➕ **Add Custom Quotes**  
🖤 **Full AMOLED Theme** with **pitch red glow outlines**
